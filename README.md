# Student Grade Analysis (C)

## Description
A menu-driven C program that analyzes student grades using arrays and conditional logic. The program allows users to input grades and perform multiple analyses on the data.

## Features
- Enter grades for multiple students
- Calculate class average
- Find highest and lowest grades
- Count passed and failed students
- Input validation for grades

## Technologies Used
- C Programming Language
- Arrays
- Loops and Conditional Statements

## How to Run
1. Compile the program using a C compiler (e.g., GCC).
2. Run the executable file.
3. Follow the menu options displayed on the console.

## Learning Outcome
- Strengthened understanding of arrays and control structures
- Improved logical thinking and menu-driven program design
